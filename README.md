# Mia for Make Beauty Store

Projeto de e-commerce desenvolvido como case de portfólio.

A Mia for Make é uma loja de maquiagem acessível com produtos a partir de R$10.

## Funcionalidades

- catálogo de produtos
- carrinho de compras
- página de produto
- layout responsivo
- integração com Instagram

## Tecnologias

Next.js
React
Tailwind CSS
